
# 22. Generate Parentheses












link:https://leetcode.com/problems/trapping-rain-water/description/

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.



 



Example1:
```bash
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]

```

Example2:
```bash
Input: n = 1
Output: ["()"]



```









Constraints:

1 <= n <= 8
